Remarks

In each method a width scale of 1.0 will mean the width is unchanged from the position of the sockets. A width scale of 0.0 will cause a trail of zero width. From Centre = Trail width is scaled outwards from the centre point between the two sockets. From First = Trail width is scaled outwards from the position of the first socket. From Second = Trail width is scaled outwards from the position of the Second socket.
